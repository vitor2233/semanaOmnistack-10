# semanaOmnistack-10

App built during omnistack 10 with Node.js, ReactJS

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/vitor2233/semanaOmnistack-10.git`

### Setup

> install npm

```shell
$ cd backend
$ npm install
$ npm start
```
